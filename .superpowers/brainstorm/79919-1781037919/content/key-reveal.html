How should a newly created key be revealed?

The raw key is only returned once by the server — this is the only moment the user can copy it.

A

Modal with copy button

After clicking "Create", a dialog appears showing the full key with a one-click copy button. User must close it manually — forcing them to acknowledge they copied it.
